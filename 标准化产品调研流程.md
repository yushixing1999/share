## 🔄 标准化调研流程

### 阶段一：广泛搜索（多引擎聚合）

**使用技能**: `multi-search-engine`

**目标**: 全面覆盖，获取多元信息源

```bash
# 1. 百度搜索（国内视角）
web_fetch "https://www.baidu.com/s?wd=产品名+市场分析"

# 2. 微信搜索（行业观点）
web_fetch "https://wx.sogou.com/weixin?type=2&query=产品名+评测"

# 3. 头条搜索（热点资讯）
web_fetch "https://so.toutiao.com/search?keyword=产品名+新闻"

# 4. Google搜索（国际视角）
web_fetch "https://www.google.com/search?q=product+review+2025"

# 5. DuckDuckGo（隐私搜索）
web_fetch "https://duckduckgo.com/html/?q=product+comparison"
```

---

### 阶段二：深度验证（AI交叉验证）

**使用技能**: `tavily-search`

**目标**: AI优化结果，验证关键信息

```bash
export TAVILY_API_KEY="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"

# 基础搜索（快速验证）
node skills/tavily-search/scripts/search.mjs "竞品功能对比" -n 10

# 深度搜索（全面研究）
node skills/tavily-search/scripts/search.mjs "行业趋势分析" --deep

# 新闻搜索（最新动态）
node skills/tavily-search/scripts/search.mjs "产品发布" --topic news --days 7

# 提取网页内容
node skills/tavily-search/scripts/extract.mjs "https://example.com/article"
```

---

### 阶段三：精准提取（结构化数据）

**使用技能**: `exa`

**目标**: 神经网络搜索，提取精准结构化数据

```bash
export EXA_API_KEY="f39a7ea6-87b3-40e9-a0d3-eea3152b9fcb"

# 网页搜索（神经模式）
bash skills/exa/scripts/search.sh "竞品分析" 10 neural

# 代码搜索（技术调研）
bash skills/exa/scripts/code.sh "开源实现" 5

# 获取详细内容
bash skills/exa/scripts/content.sh "URL1" "URL2" "URL3"

# 特定类型搜索
bash skills/exa/scripts/search.sh "公司信息" 5 auto company
bash skills/exa/scripts/search.sh "研究论文" 5 auto research-paper
```

---

### 阶段四：内容摘要（生成调研摘要）

**使用技能**: `summarize`

**目标**: 快速生成关键信息摘要

```bash
# 网页摘要
summarize "https://example.com/article" --length medium

# PDF文档摘要
summarize "/path/to/report.pdf" --length long

# 批量处理
summarize "URL1" "URL2" "URL3" --json > summary.json

# 提取关键内容（不总结）
summarize "https://example.com" --extract-only
```

---

### 阶段五：报告生成（整合输出）

**手动整合**: 基于前面收集的数据生成最终报告

## 🎯 实战示例：调研"AI代码助手"产品

### Step 1: 多引擎广泛搜索

```bash
# 国内视角
web_fetch "https://www.baidu.com/s?wd=AI代码助手+市场分析+2025"
web_fetch "https://wx.sogou.com/weixin?type=2&query=AI编程助手+对比评测"

# 国际视角
web_fetch "https://www.google.com/search?q=AI+coding+assistant+comparison+2025"
web_fetch "https://duckduckgo.com/html/?q=GitHub+Copilot+vs+Cursor"
```

### Step 2: AI交叉验证

```bash
export TAVILY_API_KEY="tvly-dev-..."

node skills/tavily-search/scripts/search.mjs \
  "AI coding assistant market share features pricing" \
  --deep -n 15
```

### Step 3: 结构化数据提取

```bash
export EXA_API_KEY="f39a7ea6-..."

# 搜索竞品公司信息
bash skills/exa/scripts/search.sh \
  "GitHub Copilot Cursor Tab features comparison" \
  10 neural

# 获取技术文档
bash skills/exa/scripts/code.sh \
  "AI code completion open source" \
  5
```

### Step 4: 生成摘要

```bash
# 对关键文章生成摘要
summarize "https://example.com/ai-coder-comparison" --length long
summarize "https://example.com/copilot-review" --length medium
```

### Step 5: 整合报告

基于收集的数据，生成最终产品调研报告。
