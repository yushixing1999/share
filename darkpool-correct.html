  • 捕捉主力隐蔽的、不易被发现的资金流向
  • 与明盘资金（公开的大单）形成互补

📚 一、什么是暗盘资金（正解）

1.1 核心定义

暗盘资金 = 主力通过拆单行为（将大单拆成小单）进行买卖的净额 拆单：将一笔大单（如1000万）拆分成多笔小单（如10万×100笔） 目的：避免被Level-2大单监控发现，隐蔽建仓/出货

关键理解：

  • 明盘资金：公开的大单交易（>100万），容易被监控
  • 暗盘资金：主力拆单后的小单交易，隐蔽性强，需要专门算法识别
  • 运行时段：交易时段内实时计算（9:30-11:30, 13:00-15:00）

1.2 为什么叫"暗盘"

对比项 明盘（公开） 暗盘（隐蔽）
单笔金额 大单（>100万） 小单（<50万）
可见性 Level-2明显可见 隐藏在小单中
主力意图 公开操作 隐蔽建仓/出货
识别难度 容易 需要专门算法

🔍 二、暗盘资金的计算原理

2.1 如何识别拆单行为

识别特征：

  1. 时间集中度：短时间内（如1分钟）出现多笔相似金额的小单
  2. 价格一致性：多笔小单以相同或相近价格成交
  3. 方向一致性：连续多笔同为买入或卖出
  4. 金额规律性：金额呈现规律性的拆分（如都是9.8万、9.9万）

2.2 指标展示形式

在同花顺软件中，暗盘资金通常以副图指标形式展示：

显示元素 含义
红色柱状 当日暗盘资金净流入（主力隐蔽吸筹）
绿色柱状 当日暗盘资金净流出（主力隐蔽出货）
柱状高度 资金流入/流出幅度，越高幅度越大
零轴 多空分界，红柱在上、绿柱在下

📊 三、实战用法

3.1 五大经典信号

🟢 信号一：暗盘吸筹（看涨）

特征： K线横盘或微跌，但暗盘资金持续为红柱（流入）

解读： 主力在低位通过拆单隐蔽吸筹，不引起市场注意

策略： 关注突破信号，准备跟进

🔴 信号二：暗盘出货（看跌）

特征： K线横盘或微涨，但暗盘资金持续为绿柱（流出）

解读： 主力在高位通过拆单隐蔽出货

策略： 警惕回调风险，考虑减仓

⚠️ 信号三：顶背离（危险）

特征： K线创新高，但暗盘资金为绿柱或红柱缩短

解读： 价格上涨但主力暗地在出货

策略： 顶部信号，考虑离场

💎 信号四：底背离（机会）

特征： K线创新低，但暗盘资金为红柱或绿柱缩短

解读： 价格下跌但主力暗地在吸筹

策略： 底部信号，关注反弹

🔄 信号五：多空分歧（观望）

特征： K线窄幅震荡，暗盘资金围绕零轴小幅波动

解读： 主力方向不明，处于观察期

策略： 观望为主，等待方向明确

⚖️ 四、与明盘资金的对比

对比维度 明盘资金（大单流向） 暗盘资金（拆单统计）
统计对象 单笔>100万的大单 聚类后的小单组合
主力意图 公开操作（可能故意为之） 隐蔽操作（真实意图）
可靠性 容易被主力利用（骗线） 更接近真实资金动向
使用场景 确认趋势、追踪公开主力 发现隐蔽建仓/出货、识别骗线
最佳实践：明盘+暗盘结合使用
  • 明盘资金净流入 + 暗盘资金净流入 → 🟢 真实上涨，可跟进
  • 明盘资金净流入 + 暗盘资金净流出 → ⚠️ 可能是骗线，谨慎
  • 明盘资金净流出 + 暗盘资金净流入 → 💎 主力隐蔽吸筹，关注
  • 明盘资金净流出 + 暗盘资金净流出 → 🔴 真实下跌，回避

📝 五、总结

正确理解：

同花顺暗盘资金是交易时段内实时运行的技术指标，用于统计主力通过"拆单"（将大单拆成小单）进行隐蔽交易的净额。

核心价值：
1. 发现主力隐蔽的建仓/出货行为
2. 识别公开资金流向（明盘）背后的真实意图
3. 与明盘资金结合，提高判断准确性

不是：盘后交易分析工具、次日开盘预测工具
